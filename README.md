A migration tool for PostgreSQL

### Get started

```bash
brew install direnv
```
### Check that direnv is working
```bash
cd hire-db
bluefin rebuild dev
```


```bash
git clone git@github.com:dailymuse/bluefin.git
cd bluefin
bin/bluefin --help
```

### Usage

import the library

`const bluefin = require('bluefin')`
